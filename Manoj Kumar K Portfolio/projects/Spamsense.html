SpamSense: Spam Email Classifier Using NLP

  • GitHub URL: Project Link

Developed a scalable spam email classification system combining classical and deep learning NLP techniques. Preprocessed large email datasets stored in MongoDB and Oracle PL/SQL databases. Leveraged Spark and Hadoop for distributed data processing and feature extraction. Built, trained, and fine-tuned classification models using TensorFlow, PyTorch, and Scikit-learn. Exposed the model via RESTful APIs developed with FastAPI in Python and integrated backend services using Java. Containerized the application with Docker and deployed on AWS, orchestrated via Kubernetes for high availability. Created interactive dashboards using Tableau and Power BI to monitor email traffic and spam trends, complemented by detailed visualizations in Seaborn and automated reporting with Excel VBA. Managed version control and CI/CD pipelines using Git.
